Restrictions

  • Only use HTML (no CSS of any kind)
  • In addition to the required html, head, title, and body elements, you can use any of the following HTML elements:
    • div
    • span
    • h1, h2, h3, h4, h5, h6
    • strong
    • em
    • ol, ul, li
  • Again you can use & n b s p ; (but without spaces) to equal a visible space.

    • Goals

    Focus on presenting the text in different hierarchies and scales using only HTML.
